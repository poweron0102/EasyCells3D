## Blender GLB/GLTF

A engine possui um `SceneLoader` para carregar cenas exportadas do Blender como `.glb` ou `.gltf`, criando `Items` com hierarquia, transforms e componentes configurados por Custom Properties.

Documentação: `docs/blender_gltf_scene_loader.md`
